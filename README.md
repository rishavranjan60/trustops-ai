# TrustOps AI

TrustOps AI is a production-grade AI evaluation and MLOps platform for building, evaluating, monitoring, and deploying Machine Learning and LLM/RAG systems.

## Core Features

- Dataset upload and validation
- ML model training
- MLflow experiment tracking
- Model comparison
- Explainable AI with SHAP
- Bias and fairness analysis
- Data drift monitoring
- LLM/RAG evaluation
- Deployment readiness scoring

## Tech Stack
